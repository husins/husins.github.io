GitHacker使用总结

2021-01-28 工具使用 242 字
git

今天我中了GitHack的诡计，他竟然还有个强大的哥哥GitHacker

GitHacker 和 Githack 有什么区别呢？

GitHacker是一个多线程工具，用于检测站点是否存在git源码泄漏，并能够将网站源代码下载到本地。值得一提的是，这个工具会将整个git repo恢复到本地

Githack，只是简单的恢复到最新版本。

如此一来，通过GitHacker你就可以查看到开发人员的历史提交记录以及提交注释，以更好的掌握开发者的性格和心理，从而为进一步的代码审计奠定基础。

它结合git命令能够产生妙用

  • python githacker.py 127.0.0.1/.git/将git泄露的文件，下载到本地
  • git log --reflog 查看提交过的版本信息和分支信息
  • git reset -–hard彻底回退到某个版本，本地的源码也会变为上一个版本的内容
